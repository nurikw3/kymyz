# Список параметров

| Параметр | Описание |
|---|---|
| `timestamp` | дата и время измерения |
| `Cu_charge` | содержание Cu в шихте |
| `Cu_matte` | содержание Cu в штейне |
| `Cu_slag` | содержание Cu в шлаке |
| `Zn_charge` | содержание Zn в шихте |
| `Zn_slag` | содержание Zn в шлаке |
| `CaO_charge` | содержание CaO в шихте |
| `CaO_slag` | содержание CaO в шлаке |
| `SiO2_charge` | содержание SiO2 в шихте |
| `SiO2_slag` | содержание SiO2 в шлаке |
| `Fe_charge` | содержание Fe в шихте |
| `Fe_slag` | содержание Fe в шлаке |
| `S_charge` | содержание серы в шихте |
| `charge_flow` | расход шихты |
| `concentrate_flow` | расход концентрата |
| `coal_flow` | расход угля |
| `coal_dust_flow` | расход угольной пыли |
| `addition_flow` | расход подшихтовочных материалов |
| `air_flow` | расход воздуха |
| `oxygen_flow` | удельный расход кислорода на тонну металлосодержащих / шихты |
| `oxygen_concentration` | концентрация кислорода в дутье |
| `natural_gas_flow` | расход природного газа |
| `offgas_temperature` | температура отходящих газов |
| `melt_temperature` | температура расплава |
| `caisson_temperature` | температура кессонов |
| `furnace_pressure` | разрежение / давление в печи |
| `offgas_flow` | расход отходящих газов |
| `SO2_out` | концентрация SO2 на выходе из печи |
| `charge_moisture` | влажность загружаемых материалов |
| `melt_level` | уровень расплава |
| `water_flow_circuits` | расход воды в контурах охлаждения |
| `feedwater_flow_boiler` | общий расход питательной воды на барабан котла-утилизатора |
| `boiler_inlet_gas_temperature` | температура отходящих газов на входе в котел |
| `steam_pressure_drum` | давление пара в барабане |
| `steam_temperature_drum` | температура пара в барабане |
